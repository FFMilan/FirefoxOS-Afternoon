Firefox OS Afternoon

Questa è la pagina ufficiale dell'evento settimanale Firefox OS Afternoon organizzato da Aronne Brivio e Niccolò Cantù che si terrà ogni mercoledì pomeriggio presso l'Università degli Studi di Milano (ottieni le indicazioni).
Il primo incontro si è tenuto il 29 maggio 2015, qui potete trovarne la relazione.

In questa trovate gli strumenti utilizzati durante gli incontri ed alcuni link utili.

Strumenti utili

  • Building Blocks - UI components per l'interfaccia di Gaia - pagina ufficiale
  • Firefox OS Simulator - Add-on per Firefox che installa il simulatore di Firefox OS sul proprio pc.
  • WebIDE (Istruzioni per l'utilizzo) - Ambiente di editing e sviluppo applicazioni per Firefox OS. In alternativa è possibile utilizzare l'App Manager, ormai sostituito dal WebIDE, a cui è possibile accedere digitando about:app-manager sulla URL bar.

Link utili

  • Firefox OS Boilerplate App - Applicazione di esempio per le principali API di Firefox OS
  • Guida veloce allo sviluppo per Firefox OS - Traduzione italiana dell'omonimo testo di Andrè Garzia a cura di Daniele Scasciafratte e Edoardo Putti
  • How to contribute - Diventa un volontario Mozilla!
  • MDN - Mozilla Developer Network: tutto il necessario per un WebApp developer
  • Mozilla IRC - Server IRC di Mozilla, #developers-it è il canale ufficiale degli sviluppatori italiani
  • Mozilla Italia - Associazione italiana di supporto e traduzione Mozilla
