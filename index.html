FirefoxOS Afternoon

Questa è la pagina ufficiale dell'evento FirefoxOS Afternoon del 29 Aprile 2015 organizzato da Aronne Brivio e Niccolò Cantù.
In questa pagina potrete trovare gli strumenti utilizzati durante l'incontro ed alcuni link utili.

Strumenti utili

  • Building Blocks - UI components per l'interfaccia di Gaia
  • FirefoxOS Simulator - Add-on per Firefox che installa il simulatore di FirefoxOS sul proprio pc.
  • Gobby - Gobby è un editor collaborativo, che permette di lavorare sugli stessi file contemporaneamente.
  • WebIDE (Istruzioni per l'utilizzo) - Ambiente di editing e sviluppo applicazioni per FIrefoxOS. In alternativa è possibile utilizzare l'App Manager, ormai sostituito dal WebIDE, a cui è possibile accedere digitando about:app-manager sulla URL bar.

Link utili

  • FirefoxOS Boilerplate App - Applicazione di esempio per le principali API di FirefoxOS
  • Guida veloce allo sviluppo per Firefox OS - Traduzione italiana dell'omonimo testo di Andrè Garzia a cura di Daniele Scasciafratte e Edoardo Putti
  • MDN - Mozilla Developer Network: tutto il necessario per un WebApp developer
  • Mozilla IRC - Server IRC di Mozilla, #developers-it è il canale ufficiale degli sviluppatori italiani
  • Mozilla Italia - Associazione italiana di supporto e traduzione Mozilla
